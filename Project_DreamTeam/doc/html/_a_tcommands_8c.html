Write a given command to buffer as provided.

Note
Newline "\r\n" is always appended to the buffer
Parameters
buffer... character buffer which will be written too. Should be big enough to hold AT_CMD_BUFFER_SIZE
tx... a callback function, which sends the buffer upon successful buffer write
cmd... an AT command template to use or a pre-made string
Returns

Write a given command to buffer, up to a given limit, with the provided parameters as suffix, limited by.

Note
Newline "\r\n" is always appended to the buffer
Parameters
buffer... character buffer which will be written too. Should be big enough to hold AT_CMD_BUFFER_SIZE
tx... a callback function, which sends the buffer upon successful buffer write
limit... an upper limit of characters to write. Writing less is possible.
cmd... an AT command template to use
param_format... a format string for the parameters provided
